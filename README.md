# Scratch-Project

## Summary
A fullstack webpage into which users can put input a skincare ingredient and see which Sephora products 
are free of those ingredients.


## Tech Stack
Programming Language:
- Javascript

Frontend: 
- React with Hooks
- Redux Tool-Kit
- CSS, Sass
- Webpack 

Backend: 
- Node
- Fetch API
- Express Framework with Routers
- SQL Database
